UoA Dessert Society Website 2025

Date:

April 2025

About:

A website created for the 2025 University of Auckland Dessert Society. This website displays information about the club itself, upcoming and past events and sponsors. The project utilizes the Notion database to keep an up to date record of sponsors and events.
After working on the UADS 2024 website in a small development team last year, I decided to use the skills I learnt and redesign and rebuild the website for 2025. As the Co-President of the club, I designed and worked on this project alone and it took about a week to complete.
The sponsor page contains a grid of our sponsors, of which all images link to the sponsor's website. The sponsor cards all contain information about the deals some of our sponsors are offering. There is also a search function to find the sponsor faster.
The event page contains a list of our events, sorted by upcoming events closest in date to furthest in date, followed by past events closest in date to furthest in date. Upcoming event cards have a red background, which differentiates them from past events which are shown with a red border.
There are unique event pages which contain the full description of the events and an embedded sign up form if the event has not occured yet. These follow similar patterns where past events have a red outline while upcoming events have a red background. They also contain links to the instagram post.
The about page just displays information about the club and contains a grid of all the execs, which is taken from the notion CMS.
Note: I utilised AI to assist with backend development as I did not manage to learn last year.

Tools and Languages:

React, TypeScript, TailwindCSS, Figma, NotionCMS, Express

Status:

Complete and Deployed

Github:

uads-2025-app

Project Website:

uads.co.nz

Topsterify Spotify API

Date:

July 2024

About:

A React webapp that utilizes the Spotify API to show a user's top artists and tracks in a visual format. Users are given the choice to look at their top data from 3 distinct periods: last 4 weeks, 6 months and 12 months. The webapp is also responsive with dedicated views for different screen sizes. I worked on this project alone as music is one of my main interests and I plan on updating it with new functionality in the future.

React, JavaScript, TailwindCSS, Figma
